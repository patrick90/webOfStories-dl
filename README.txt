HOW TO USE:
First,  make sure you have BeautifulSoup installed.

You can install it with

pip install beautifulsoup4

or

easy_install beautifulsoup4.

For more info, see http://www.crummy.com/software/BeautifulSoup/#Download.

Two, how to use.
Let say you want to downloand transcript of all stories of Marvin Minsky from webofstories, then you should write this in console:
python webos.py http://www.webofstories.com/play/marvin.minsky/

In this same folder that you run this script there will be created a html file, with all transcripts. In this example: marvin.minsky.html
